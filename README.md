# TADC — The AI-Driven Character

TADC est un jeu sandbox 2D en Pixel Art où un personnage autonome piloté par une intelligence artificielle explore, décore et vit sa propre vie. L'IA prend toutes les décisions : déplacements, placement de meubles, exploration, et peut même parfois invoquer des amis.

## Fonctionnalités

- **Personnage piloté par l'IA** : Observez le personnage prendre vie de manière autonome.
- **Génération par IA** : Générez dynamiquement des personnages et des meubles grâce à l'IA.
- **Mode Observation** : L'IA décide de tout, vous êtes le spectateur de son quotidien.
- **Support Multi-IA** : Compatible avec Google Gemini et Mistral AI (clé API requise).
- **Interface Intégrée** : Suivez en temps réel les pensées (💭) et les actions (⚡) de l'IA via le panneau latéral.

## Technologies Utilisées

- HTML5 (Canvas)
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Intégration API LLM (Gemini, Mistral)

## Configuration et Lancement

1. Clonez ou téléchargez ce projet.
2. Ouvrez le fichier `index.html` dans un navigateur web moderne. (Vous pouvez également utiliser un serveur de développement local comme Live Server pour une meilleure expérience).
3. Sur l'écran titre, choisissez votre fournisseur d'IA (Google Gemini ou Mistral AI) et sélectionnez le modèle.
4. Entrez votre clé API valide pour le fournisseur choisi.
5. Cliquez sur "Sauvegarder la clé" puis sur "Lancer le jeu".

## Guide d'Utilisation

Une fois dans le jeu, utilisez le panneau latéral pour interagir et observer :
- **Générer le Personnage** : Invoquer votre premier habitant géré par l'IA.
- **Générer Meuble IA** : Demander à l'IA de créer et placer des éléments de décoration dynamiques.
- Suivre les pensées et actions du personnage en temps réel.
- Mettre en pause ou relancer l'IA si nécessaire.
